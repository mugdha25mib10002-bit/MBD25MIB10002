<h1># Problem Statement
</h1>
<br>
<h5>People often require a reliable way to set alarms for waking up, reminders, and daily tasks. Traditional clocks can be rigid and lack customizability. There is a need for a simple, software-based alarm clock that is easy to use and provides accurate alerts.
<br>

-------------------------------------------------------------------------

<br>
<h2># Scope of the Project</h2>
<br>
<h5>The Smart Alarm Clock System allows users to:</h5>

- Set a specific time for an alarm.  
- Check and save the alarm time.  
- Sound the alarm exactly at the chosen time.  
- Use a clean and simple interface.
- <br>
The project includes GUI design, multi-threading, and time-based event handling.
<br>

-----------------------------------------------------------------------------

<br>
<h2># Target Users</h2>

- Students who need study reminders  
- Working professionals who need task alerts  
- Anyone who needs a customizable computer-based alarm  
- Users who want a simple desktop interface
- <br>

-------------------------------------------------------------------------------

<br>
<h2># High-Level Features</h2>

- Time input fields (hour, minute, second)  
- Start alarm button  
- Real-time monitoring in the background  
- Sound alert when time matches  
- Error handling for invalid input  
- Smooth GUI without freezing, using threading
